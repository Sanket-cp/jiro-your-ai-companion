# Jiro - AI Voice Assistant

A sophisticated AI voice assistant inspired by Jarvis with a beautiful futuristic interface.

## Features

- Voice and text chat with AI
- Multilingual support (English, Hindi, Bengali)
- Modern, responsive UI
- Real-time conversation

## Deployment on Vercel

1. Push your code to GitHub
2. Connect your GitHub repo to Vercel
3. Set up environment variables in Vercel dashboard:
   - `OPENAI_API_KEY` - Your OpenAI API key
4. Deploy!

## Environment Variables

For the Supabase function to work, you need:
- `OPENAI_API_KEY` - Get this from OpenAI's website

## Local Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The build output will be in the `dist` folder, ready for deployment.
